# upload-for-vue

|传入的参数  | 意义 |类型|
|:------------- |:---------------:| -------------:|
name  | 上传的name属性值|String
mutifile  |多文件上传|Boolean(默认false)
autoupload |自动上传|Boolean（默认false）
type |支持的文件后缀|String
address|上传地址|String
inputCallFun |选择文件后的回调|Function
sucCallFun |上传成功后的回调|Function
falCallFun |上传失败后的回调|Function
